CSS Selectors

Selector Tag Description
Type article {} You can select a type of HTML tag by simply writing the tag.
Class .content {} Select a html class by prepending a . to the beginning.
ID #unique-ID An ID attribute can be selected by prepending a # to the beginning.
Chaining Selectors article.content {} You can chain selectors to be more specific in applying a CSS rule. The example shown here will only apply the rule to items with a class of "content" that also have an "article" tag.
CSS Descendant article h2 This will only style "h2" elements that are descendants of an article tag.
Multiple CSS Selectors article, h2, .content {} You can use multiple CSS selectors for the same rule by separating the selectors with commas.
